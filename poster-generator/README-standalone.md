# 网易游戏玩家招募海报生成器 - 独立版## 文件说明- **[player-recruitment-poster-standalone.html](file:///D:/公众号招募/poster-generator/player-recruitment-poster-standalone.html)** — 完整可独立运行的招募海报生成工具（约 491KB）## 使用方法1. 双击 `player-recruitment-poster-standalone.html` 在浏览器中打开2. 在左侧控制面板填写：   - 城市名称、活动日期   - 目标玩家要求   - 活动时间段、时长、地点   - 活动任务描述   - 礼金金额（基础报酬 + 交通补贴）   - 报名提示文字   - 上传报名二维码图片（PNG/JPEG）   - 项目编号   - 社群引流文字   - 上传社群二维码图片3. 点击 **"生成海报"** 按钮实时预览4. 点击 **"下载 PNG"** 导出高清海报图片5. 点击 **"打印/另存为"** 打印或保存## 功能特点- **纯本地运行** — 不需要服务器，不依赖网络- **单文件设计** — 所有样式、脚本、图片素材全部内置- **实时预览** — 左边输入，右边海报即时更新- **二维码上传** — 支持用户上传报名和社群二维码- **高清导出** — 使用 html2canvas 实现 2 倍分辨率导出- **打印支持** — 原生打印功能，控制面板自动隐藏## 注意事项- 请使用现代浏览器（Chrome/Edge/Firefox）- 二维码上传支持 PNG 和 JPEG 格式- 所有表单均含默认演示数据，可直接替换

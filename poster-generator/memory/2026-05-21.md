## PNG Download Missing Decorations (2026-05-21 11:46)
- User reported downloaded PNG is missing decorative elements
- Root cause: `downloadPNG` function in `player-recruitment-poster-decorated.html` explicitly removes elements with `.deco` classes before html2canvas rendering
- This was originally done to prevent CORS/tainted canvas issues with external or local image paths
- Since all decoration images are now embedded as base64 data URIs, the `.deco` removal step is obsolete
- Fix: Remove/comment out the `.remove()` logic that strips `.deco` elements during canvas clone
- Files involved: `player-recruitment-poster-decorated.html`, `.cowork-temp\images_b64.js`, `.cowork-temp\inject_b64.py`
